# 高速公路服务区智能化场景演示

本项目包含帆软为高速公路服务区设计的智能化场景演示材料。

## 产出物清单

### 1. 演示PPT
- `服务区智能化场景.pptx` - 三页演示幻灯片
  - 公众服务场景
  - 安全管理场景  
  - 满意度大模型场景

### 2. 运营工具
- `服务区日报模板.xlsx` - 服务区日常运营日报模板
  - 日报主表
  - 填写说明
  - 汇总看板

### 3. 智能问答Demo
- `服务区智能问答系统V2.html` - **新版** 全场景智能问答系统
  - 5大问答场景：路线导航、停车查询、餐饮推荐、加油服务、景点推荐
  - 实时数据面板：服务人数、停车占用率、加油排队等
  - 服务状态监控：停车场、加油站、餐厅、卫生间状态
  - 热门查询推荐：最近服务区快速导航
- `服务区智能问答Demo.html` - APP智能问答交互演示（V1版本）
- `服务区区长秘书大模型Demo.html` - 区长秘书助手演示
- `服务区绩效考核大模型Demo.html` - 绩效考核智能分析演示
- `服务区预警指挥调度Demo.html` - 预警指挥调度演示

### 4. 场景说明文档
- `服务区预警智能调度大模型场景说明.md` - 预警智能调度场景详细说明

## 使用方式

HTML Demo 文件可直接在浏览器打开体验交互效果。

PPT 和 Excel 文件使用 Microsoft Office 或 WPS 打开。

## 技术栈

- 帆软 FineReport / FineBI
- 大语言模型 (LLM)
- 语音识别 / 语音合成
